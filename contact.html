Questions

If you have a question about code and are seeking my help with it, please don’t send it via email—create a reduced test case, open a Stack Overflow thread instead and share the link with me via Twitter. I’ll do my best to answer the question if and whenever I can.

Questions answered on Stack Overflow will still be available there for others who face the same issues to find the answer to in the future, and thus would be a contribution to the community as well.

I apologize in advance for not answering troubleshooting requests via email unless you’re hiring me to do so.
